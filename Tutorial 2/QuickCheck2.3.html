Questions

  1. Provide a style rule to display all unordered lists with lowercase letters as the list markers.
  2. Provide a style rule to display all unordered lists using the star.png image file, placed inside a virtual box.
  3. Provide a sryle rule to display the text of all previously visited hypertext links in grey.
  4. Provide the style rule to set the padding around every h1 heading in a section element to 1em on top 0.5em on the left and right, and 2em on the bottom.
  5. Provide the style rule to change the left margin of the figure element to 20 pixels.
  6. Describe the item selected by the folowing selector;

    #top > p:first-of-type:first-line

  7. Describe the items selected by the following selector

    div.Links imp[usemap]

  8. Provide the style rule to insert the text string "***" before every paragraph belonging to the Review class.
  9. Provide the style property to set the opening quotation mark and closing quotation marks to curly quotes with unicode values of 2018 and 2019 respectively.

Answers

  1. ul{list-style-type: lower-alpha;}
  2. ul {list-style-image: url(start.png);list-style-position: inside;}
  3. a:visited {color : gray;}
  4. section h1 {padding: 1em 0.5em 2em;}
  5. figure {margin-left: 20px;}
  6. The first line of the first paragraph that's within the element id = "top"
  7. All image elements that have the usemap attribute within the div element belonging to the Links class.
  8. p.Review::before {content: "***";}
  9. quotes:"\ߢ" "\ߣ"
